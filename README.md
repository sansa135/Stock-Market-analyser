# 📈 Stock Market Analytics System

A Python-based Stock Market Analytics System that fetches real-time stock data using API and performs analysis with interactive graph visualization.

---

## 🚀 Features

- 📊 Live stock data using API
- 📉 Trend detection (Uptrend / Downtrend)
- 📈 Moving Average calculation
- 📌 Highest & Lowest price analysis
- 📊 Graph visualization using Tkinter (no external library)
- 🔥 Multi-stock comparison (2 stocks)

---

## 🛠️ Technologies Used

- Python
- API Integration (Alpha Vantage)
- Tkinter (Graph Visualization)
- Data Analytics Logic
